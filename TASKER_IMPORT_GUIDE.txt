================================================================================
TROUBLESHOOTING
================================================================================

**Automation doesn't start?**
→ Make sure AutoInput Accessibility is enabled
→ Open AutoInput app and check "Service Enabled" status

**Taps are missing buttons?**
→ Coordinates may be wrong for your screen
→ Edit task and adjust ±10 pixels

**Keys not being typed?**
→ AutoInput needs permission to type
→ Check AutoInput settings

**Screen turns off during testing?**
→ Change screen timeout to "Never"
→ Or plug in charger

**App crashes during testing?**
→ Check if SE app is up to date
→ Clear app cache and try again

**Progress seems stuck?**
→ Tasker is still running
→ Check Tasker → Active tasks
→ Wait for it to complete

================================================================================
WHAT'S IN THE TASKER PROJECT
================================================================================

**Profile:** SE Auto Key Test
**Trigger:** Can be run manually (tap Play button)
**Task:** Test All 832 Keys

**Actions (19 steps):**
1. Initialize KeyIndex = 1
2. Store all 832 keys in %AllKeys variable
3. Loop 832 times:
   - Extract current key from list
   - Wait 2 seconds
   - Launch SE app
   - Wait 2 seconds
   - AutoInput: Click Entry Key (540,943)
   - Wait 2 seconds
   - AutoInput: Click Key Field (540,541)
   - Wait 1 second
   - AutoInput: Write %CurrentKey
   - Wait 1 second
   - AutoInput: Click Activate (834,657)
   - Wait 5 seconds
   - Take screenshot
   - AutoInput: Click Close (474,608)
   - Wait 1 second
   - Kill SE app
   - Increment KeyIndex
   - Progress notification (every 50 keys)
   - Loop back

================================================================================
ALL 832 KEYS EMBEDDED
================================================================================

The Tasker project contains all 832 keys embedded in the %AllKeys variable:

000000,017C0A,017C0a,017c0A,017c0a,0XUEKN,0XUEKn,...,xnkjum

Each key is extracted one by one using string splitting.
