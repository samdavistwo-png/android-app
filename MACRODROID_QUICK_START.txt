================================================================================
TROUBLESHOOTING
================================================================================

Macro doesn't work?
→ Check Accessibility is enabled for MacroDroid
→ Settings → Accessibility → MacroDroid → ON

Taps missing buttons?
→ Adjust coordinates ±10 pixels if needed

Keys not pasting?
→ Make sure you copied the key first
→ Try "Type Text" instead of "Paste"

Screen locks during testing?
→ Settings → Display → Sleep → Never
→ Or keep phone plugged in
