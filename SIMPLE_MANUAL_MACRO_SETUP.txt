================================================================================
SIMPLE MANUAL MACRO SETUP - Since Import Doesn't Work
================================================================================

MacroDroid's import feature isn't working properly. Instead, let's create a
SIMPLE macro manually that you can use to test keys quickly.

This is a SEMI-AUTOMATIC helper - you copy each key, macro runs automatically.

================================================================================
STEP 1: CREATE NEW MACRO
================================================================================

1. Open MacroDroid
2. Tap the "+" button (bottom right)
3. Name it: "Quick Key Test"
4. Tap "Add Trigger"

================================================================================
STEP 2: ADD TRIGGER (Volume Down × 3)
================================================================================

1. Select: "Hardware Key"
2. Choose: "Volume Down"
3. Pattern: "Triple Press" (3 quick presses)
4. Screen state: "Any"
5. Tap "OK"

================================================================================

- Tap "OK"

================================================================================

3. Done!

================================================================================
TROUBLESHOOTING
================================================================================

Volume Down doesn't trigger?
→ Make sure macro toggle is ON (enabled)
→ Try 3 QUICK presses (within 1 second)
→ Check Accessibility is enabled for MacroDroid

Macro runs but taps wrong spots?
→ Adjust coordinates in Actions 4, 6, 10, 13
→ Your screen: 1080 x 2400 pixels
→ Try ±10 pixels if needed

Key not pasting?
→ Make sure you copied the key first
→ Check clipboard has the key
