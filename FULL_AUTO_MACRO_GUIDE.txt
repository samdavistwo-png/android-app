================================================================================
WHAT THIS MACRO DOES
================================================================================

This macro contains ALL 832 keys embedded inside it and will:

1. Automatically loop through ALL 832 keys
2. Test each key one by one
3. Take screenshots of each result
4. STOP automatically when a working key is found
5. Alert you with sound + notification when successful
6. Show progress notifications every 50 keys

NO MANUAL COPYING! Just press Volume Down 3x and let it run for 3-4 hours!

================================================================================
HOW IT WORKS
================================================================================

TRIGGER:
- Volume Down button × 3 (quick presses)

PROCESS:
For each of the 832 keys:
1. Opens SE 2.2.6 app
2. Taps "Entry Key" button
3. Taps key input field
4. Types the key automatically (from embedded list)
5. Taps "Activate" button
6. Waits 5 seconds for validation
7. Takes screenshot
8. Checks if key was successful (OCR detection)
9. If SUCCESS → Plays sound + Shows notification + STOPS
10. If FAIL → Taps Close button + Closes app + Next key
11. Every 50 keys → Shows progress notification

TOTAL TIME: ~3-4 hours (15 seconds per key × 832 keys)

================================================================================
TROUBLESHOOTING
================================================================================

Macro not starting?
→ Check Accessibility is enabled for MacroDroid
→ Settings → Accessibility → MacroDroid → ON

Taps missing buttons?
→ Adjust coordinates ±10 pixels if needed
→ Edit macro actions manually

Keys not typing correctly?
→ Enable "Type slowly" option (already enabled)
→ Increase delay between characters (default: 100ms)

Screen turns off during testing?
→ Enable "Keep screen on" in Developer Options
→ Or plug in charger

Macro stops unexpectedly?
→ Check battery level
→ Check if phone locked
→ Check if app crashed

Success not detected?
→ Manually check screenshots in Pictures/KeyTests/
→ Look for green checkmark or "valid" message
→ OCR might miss some success messages
